NOSSA SOLUÇÃO

Buscando trazer informação ao úsuario de maneira prática e buscando automatizar o processo de coleta de dados e analise dos mesmos.

Utilizando sistemas embargados podemos criar um robo para realizar a coleta de dados e interpretação delas, assi

TECNOLOGIAS

Utilizando da experiência que tivemos no nosso primeiro semestre, conseguimos prototipar nosso projeto por meio de várias tecnologias. Sendo elas HTML/CSS, Java Script, Python e Arduino

OBJETIVOS

O principal objetivo da nossa proposta é ajudar o público tanto da pesca quanto dos pesquisadores, trazendo informações de maneira automatizada e direta ao úsuario.

Web dev

No momento estamos trabalhando com um crud básico para inserir os dados coletados pelo Arduino, atualmente ainda é necessário que um úsuario busque as informações coletadas e as registre em nosso site, registrando em uma forma de lista com data, nome de úsuario e os dados de analize

Python

Utilizando o python buscaremos processar esses dados plotalos e assim retornar para o usuario as informações devidas

Arduino

O Arduino será nosso coração do projeto com ele construiremos um sistema embarcado que funcionara como um barco automatico, realizando a coleta dos dados da água
